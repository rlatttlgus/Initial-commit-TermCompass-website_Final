# TermCompass — 투자 판단의 근거를 찾다

투자 중급자를 위한 AI 어시스턴트. 144개 검증된 투자용어 데이터베이스.

## 🚀 로컬에서 실행하기

1. 저장소 클론
```bash
git clone https://github.com/yourname/termcompass.git
```

2. Live Server 확장 설치 (VS Code)
   - VS Code Extensions에서 "Live Server" 검색 후 설치

3. HTML 파일 우클릭 → "Open with Live Server"
   - 자동으로 `http://127.0.0.1:5500/`에서 열림

## 📋 파일 구조
- `주식용어집.html` — 메인 페이지 (단일 파일)
- 외부 의존성 없음 (CDN에서 폰트/라이브러리 로드)

## 🔧 주의사항

- 현재 iframe 비활성화된 상태
